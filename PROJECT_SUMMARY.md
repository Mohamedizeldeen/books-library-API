## 🚀 How to Run

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the server:**
   ```bash
   npm start
   ```

3. **Access the application:**
   - **Web Interface:** http://localhost:3001
   - **API Base URL:** http://localhost:3001/api

4. **Run comprehensive tests:**
   ```bash
   node test_all_endpoints.js
   ```
